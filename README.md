# 🖥️ UA1 – Introdução à IHC

## Objetivo

Aplicar conceitos de Interação Humano-Computador para melhorar a comunicação entre sistema e usuário.

## Solução Desenvolvida

### Confirmação de Exclusão de Usuário

**Título:** Confirmar Exclusão

**Mensagem:**

> Tem certeza que deseja excluir o usuário **[Nome do Usuário]**?
>
> Esta ação não poderá ser desfeita.

**Botões:**

* ❌ Excluir
* ↩️ Cancelar

## Benefícios

* Comunicação mais clara;
* Redução de erros;
* Maior confiança do usuário;
* Melhor experiência de uso.

# 👕 Projeto PH StreetWear

## Descrição

O **PH StreetWear** é uma aplicação web voltada para divulgação e comercialização de produtos do segmento streetwear.

O projeto foi desenvolvido aplicando conceitos de:

* Usabilidade;
* Acessibilidade;
* Design Responsivo;
* Experiência do Usuário.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* VLibras
* Git
* GitHub

---

## 🛒 Funcionalidades

* Catálogo de produtos;
* Carrinho de compras dinâmico;
* Página de compra;
* Navegação entre páginas;
* Interface responsiva;
* Recursos de acessibilidade.

---

## ♿ Recursos de Acessibilidade

### 🔍 Aumento e Redução de Fonte

Permite ajustar o tamanho do texto conforme a necessidade do usuário.

### 🎨 Modo Daltônico

Melhora a visualização para usuários com dificuldades na distinção de cores.

### 🤟 Integração com VLibras

Traduz conteúdos para a Língua Brasileira de Sinais (Libras).

### 👴 Modo Idoso

Amplia:

* Fontes;
* Botões;
* Espaçamentos.

### 🔎 Lupa Inteligente

Amplia conteúdos da tela acompanhando o cursor do usuário.

## ▶️ Como Executar

### GitHub Codespaces

1. Abra o repositório no GitHub;
2. Clique em **Code → Codespaces → Create Codespace on Main**;
3. Aguarde o carregamento do ambiente;
4. Execute:

```bash
python3 -m http.server 8000
```

5. Abra a porta **8000** na aba **Ports**.
